# Objetivo
Repositório pessoal dos meus arquivos de configuração para facilitar eventuais migrações.

# Uso
## Realizar Backup
Basta executar 
```sh
./update_dotfiles.sh
```

Se não puder, execute `chmod +x update_dotfiles.sh` para dar permissão de execução ao script.
